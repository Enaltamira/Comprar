# Comprar

## Installation

To install the Comprar application, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/Enaltamira/Comprar.git
cd Comprar
npm install
```

## Setup

After installing the dependencies, you may need to set up environment variables. Create a `.env` file in the root directory with the following content:

```.env
NODE_ENV=development
PORT=3000
DATABASE_URL=<your_database_url>
```

Make sure to replace `<your_database_url>` with the actual connection string for your database.

## Usage

To start the application, run:

```bash
npm start
```

Visit `http://localhost:3000` in your browser to access the application.
